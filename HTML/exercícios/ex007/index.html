Listas ordenadas

  1. A tag <ol> abre uma lista ordenada
  2. este modo de lista coloca os números automáticos
  3. Além disso o parâmetro "type=" define o tipo de lista
  4. E o parâmetro "start=" define o ponto de partida da lista

Listas não ordenadas

  • não segue uma ordem
  • o parâmetro "type=" funciona aqui
