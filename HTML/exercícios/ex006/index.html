Principais formatações de texto

Negrito

Nessa frase veremos um termo em negrito usando a tag <b>

Nessa frase veremos um termo em destaque usando a tag <strong>

Itálico

Nesta frase temos um termo em itálico com a tag <i>

Nesta frase veremos um texto em ênfase usando a tag <em>

Recomendação do Guanabara

Comando para "envelopar" textos, ou seja, colocar tags, CTRL + SHIFT + P, enter + comando

É recomendável que use as tags <strong> e <em> para representar negrtito e itálico

Texto marcado

Podemos escrever um texto marcado usando a tag <mark>

Além disso vc pode mudar a cor da marcação usando css com a tag style=

Texto Grande e pequeno

Também podemos criar textos grandes (obsoleto) e textos pequenos na página

Texto deletado

podemos marcar um texto como excluído, para dizer que deve ser lido mas não considerado com a tag <del>

Texto inserido

podemos marcar um texto como inserido, para dar ênfase e indicar que ele foi adicionado com a tag <ins>

Texto sobrescrito e subscrito

Podemos usar para escrever coisas como x20+3 ou do tipo H2O com a tag <sup> e <sub>
